# Sari-Sari Store

## Getting Started

Follow these steps to run the project locally:

1. **Clone the Repository:**

    ```bash
   git clone https://github.com/rjglozano/SariSari_Store.git
    ```

2. **Install Dependencies:**

    ```bash
    npm install
    ```

3. **Run the Application:**

    ```bash
    npm run dev
    ```

4. **Run Tests:**

    ```bash
    npm test
    ```
